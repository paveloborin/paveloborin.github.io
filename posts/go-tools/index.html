Инструменты для форматирования и анализа кода в Go

June 4, 2018
go tools goimports errcheck gometalinter

Форматирование импортов

Использование:

$ goimports -d -w $(find . -type f -name ‘.go’ -not -path “./vendor/”)

Установка: $ go get golang.org/x/tools/cmd/goimports

Автоформатирование кода

Использование:

$ go fmt ./…

Анализ корректной обработки ошибок

Использование:

$ errcheck ./…

Установка:

$ go get -u github.com/kisielk/errcheck

Анализ кода по множеству параметров

Использование:

$ gometalinter ./… –exclude=vendor

Установка:

$ go get -u gopkg.in/alecthomas/gometalinter.v2
$ gometalinter –install
